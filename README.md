# children-english-ai-assistant

儿童英语AI助手模拟，包含对话与简单评估

项目经历：我曾参与“儿童英语启蒙AI助手”小程序项目。作为核心成员，我负责设计基于大模型的智能对话与纠音功能。通过分析用户对话数据，我提出优化提示词工程以提升互动相关性，推动开发上线A/B测试。新版本使平均单次使用时长提升15%，用户次日留存率提升20%。项目过程中我独立完成了竞品分析报告，并持续跟踪GPT-4V等多模态模型进展，为产品视觉问答功能规划提供了建议。

## 运行
```bash
pip install -r requirements.txt
python main.py
```
